Tentang HTML

Mengenal Elemen HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk menyusun konten di web. Elemen-elemen seperti paragraf, gambar, dan tautan mendefinisikan struktur halaman.

Apa itu HTML?

HTML bukanlah bahasa pemrograman; ini adalah bahasa markup. Ini berarti HTML menggunakan serangkaian tag dan elemen untuk memberi label dan menyusun bagian-bagian konten. Peramban web (seperti Chrome, Firefox, atau Safari) membaca dokumen HTML dan menerjemahkan tag-tag ini menjadi tampilan visual yang dapat kita lihat.

Atribut HTML

Atribut digunakan untuk memberikan informasi tambahan tentang suatu elemen. Atribut selalu ditempatkan di tag pembuka dan biasanya terdiri dari pasangan nama="nilai".

Contoh:

Dalam contoh di atas, src, alt, dan width adalah atribut yang memberikan nilai sumber gambar, deskripsi alternatif, dan lebar gambar.
